# email-ku
Simple Send Email Using Sendgrid API

## Aplication / Framework
- Spring Boot
- Sendgrid API

## Step to Setting Up
- Set API key Sendgrid in ```application.yml ``` file
- lombok annotation library

## Example request
```
{
	"emailFrom":"emailfrom@gmail.com",
	"target":"target_email@gmail.com"
	
}
```
